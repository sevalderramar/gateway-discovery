# Requerimientos

El sistema Biblioteca Online debe permitir consultar informacion basica del dominio de biblioteca mediante microservicios independientes.

## Requerimientos funcionales

- Consultar libros disponibles.
- Consultar usuarios registrados.
- Consultar prestamos.

## Requerimientos de arquitectura

- Separar responsabilidades en microservicios.
- Usar Eureka como registro de servicios.
- Usar Gateway como punto unico de entrada.
